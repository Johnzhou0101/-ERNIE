# ERNIE-Recruiting
文心大模型ERNIE在寻找AGI人才


-Coding方向
【岗位定位】
重塑模型逻辑基石： 赋予大模型顶级的逻辑推理能力，攻克代码生成、补全、Debug及仓库级（Repo-level）代码理解难题。
打造Agent核心引擎： 确立大模型在生产力工具领域的绝对壁垒，通过代码能力驱动复杂的Agent系统执行。
持续提升长文建模的能力并提升token效果和能力（成本问题）
【任职要求】
深入理解代码大模型演进路线，主导过业界领先的Code LLM训练，熟悉代码数据（含合成数据）的分布与构造。
具备扎实的编译原理、AST解析、程序分析（PL）背景，能建立代码执行与模型训练的反馈回路。
对长上下文（Long-Context）在代码场景中的应用及检索增强（RAG）代码生成有成熟落


-Agent方向
【岗位定位】
解决大模型执行复杂任务时的多步规划（Planning）、工具调用（Tool Use）与状态记忆问题，赋予模型真实的端到端执行力。
解决Agent系统在实际业务应用中的鲁棒性问题，处理环境反馈纠错、死循环阻断及多智能体协同卡点。
促进大模型和agent生态的协调发展；
【任职要求】
深入掌握ReAct、CoT等推理范式，主导过企业级Agent或复杂RAG系统从0到1的架构设计与业务落地。
具备模型层面的改造经验，能通过指令微调或强化学习针对性提升模型Function Calling的准确率与容错率。
具备强工程实现能力，精通并能深度改造现有Agent框架，解决高并发调用、状态丢失与调用延迟等工程瓶颈。
